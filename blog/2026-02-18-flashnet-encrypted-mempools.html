Executive Summary

Flashbots has introduced Flashnet, a network anonymized mempool protocol representing a significant advance in MEV protection and censorship resistance.

The Problem: Transparent Mempools

Blockchain mempools are public broadcast channels, enabling:

  • Front-running: Bots see transactions before inclusion
  • Sandwich attacks: Bots bracket user trades for profit
  • Censorship: Validators selectively exclude transactions

Flashnet Architecture

Network-level anonymity for transaction submission:

User Transaction → Encrypted Broadcast → Flashnet Protocol
                                              ↓
                    Validators receive without:
                    - Sender identification
                    - Content visibility (until inclusion)
                    - Mempool observability
